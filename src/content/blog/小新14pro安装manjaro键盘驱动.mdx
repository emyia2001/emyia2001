# 1.编辑grub文件

```
sudo gedit /etc/default/grub
```

# 2.修改成以下参数

```
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash i8042.direct i8042.dumbkbd"
```

# 3.更新grub文件&&reboot

```
sudo update-grub
sudo reboot
```
