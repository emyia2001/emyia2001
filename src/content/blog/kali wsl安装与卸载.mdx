# Install

## 安装WSL

```
wsl --install
```

### 在 WSL2 中安装 Kali Linux

以管理员身份打开 PowerShell 并运行：

```
Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux
```

重启后以管理员身份打开 PowerShell 并运行：

```
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
```

从这里下载并安装 WSL2 Linux 内核：[https ://aka.ms/wsl2kernel](https://aka.ms/wsl2kernel)
以管理员身份打开 PowerShell 并运行： wsl --set-default-version 2
从 Microsoft Store 安装 Kali Linux注意：要升级现有的 WSL1 kali-linux 安装，请键入： wsl --set-version kali-linux 2

### 安装 Win-KeX

```
sudo apt update
sudo apt install -y kali-win-kex
```

## 运行 Win-KeX

### Win-KeX 支持三种模式：

Win-KeX Win
要在具有声音支持的窗口模式下启动 Win-KeX，请运行:

```
kex --win-s
```

Win-KeX ESM
要在具有声音支持和 arm 解决方法的增强会话模式下启动 Win-KeX，请运行:

```
kex --esm --ip -s
```

要在具有声音支持的无缝模式下启动 Win-KeX，请运行:

```
kex --sl -s
```

工具包

```
sudo apt install -y kali-linux-large
```

# 卸载

### 卸载wsl子文件夹

打开cmd

```
wslconfig /l
# 从列表中选择要卸载的发行版(例如Ubuntu)并键入命令
wslconfig /u Ubuntu
```
